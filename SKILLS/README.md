# Claude Skills Collection

这个目录包含了本项目中使用的各种 Claude Skills。

## 技能列表 (Skills List)

### 1. [Frontend Design](frontend-design/SKILL.md)
- **描述**: 用于创建高质量、生产级别的个性化前端界面，避免“AI化”的设计审美。
- **许可证**: [Apache License 2.0](frontend-design/LICENSE.txt)
- **来源声明**: 该技能来源于 Anthropic 官方 GitHub 仓库：[anthropics/skills/frontend-design](https://github.com/anthropics/skills/tree/main/skills/frontend-design)。根据 Apache 2.0 许可证条款，本项目在此申明其来源，并保留原始许可证。

### 2. [Skill Builder](skill-builder/SKILL.md)
- **描述**: 专门用于构建、审核和优化 Claude Skills 的专家级助手。
- **许可证**: MIT

### 3. [Cache Components](cache-components/SKILL.md)
- **描述**: 针对 Next.js Cache Components 和局部预渲染 (PPR) 的专家级指南。
- **许可证**: [MIT License](https://github.com/vercel/next.js/blob/canary/LICENSE)
- **来源声明**: 该技能来源于 Vercel Next.js 官方仓库：[vercel/next.js/cache-components](https://github.com/vercel/next.js/tree/canary/.claude-plugin/plugins/cache-components/skills/cache-components)。

### 4. [Frontend Code Review](frontend-code-review/SKILL.md)
- **描述**: 针对 Dify 前端项目的代码审查技能，涵盖代码质量、性能和业务逻辑。
- **许可证**: [Apache License 2.0](frontend-code-review/LICENSE.txt)
- **来源声明**: 该技能来源于 Dify 官方 GitHub 仓库：[langgenius/dify/frontend-code-review](https://github.com/langgenius/dify/tree/main/.agents/skills/frontend-code-review)。

### 5. [Code Reviewer (Gemini)](gemini-code-reviewer/SKILL.md)
- **描述**: Google Gemini 官方推荐的代码审查技能，适用于本地开发和远程 PR 审查。
- **许可证**: [Apache License 2.0](gemini-code-reviewer/LICENSE.txt)
- **来源声明**: 该技能来源于 Google Gemini 官方仓库：[google-gemini/gemini-cli/code-reviewer](https://github.com/google-gemini/gemini-cli/tree/main/.gemini/skills/code-reviewer)。

### 6. [Find Skills (Vercel)](find-skills/SKILL.md)
- **描述**: 帮助发现和安装来自开放代理技能生态系统（skills.sh）的技能。
- **许可证**: [MIT License](find-skills/LICENSE.txt)
- **来源声明**: 该技能来源于 Vercel Labs 官方仓库：[vercel-labs/skills/find-skills](https://github.com/vercel-labs/skills/tree/main/skills/find-skills)。
