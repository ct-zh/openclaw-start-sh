## OpenClaw 钉钉多账号配置

用户询问钉钉多渠道部署。确认钉钉插件支持多账号配置：

```json
{
  "channels": {
    "dingtalk": {
      "enabled": true,
      "accounts": {
        "bot1": {
          "name": "钉钉机器人1",
          "clientId": "...",
          "clientSecret": "...",
          "corpId": "...",
          "agentId": "...",
          "dmPolicy": "open",
          "groupPolicy": "open"
        },
        "bot2": {
          "name": "钉钉机器人2",
          ...
        }
      }
    }
  }
}
```

结构类似飞书的 accounts 配置方式。

## 新增安全规则

用户要求添加 OpenClaw 操作限制规则，已写入 `AGENTS.md`：

- 禁止修改 OpenClaw 配置文件（如 `openclaw.json`）
- 禁止执行 `openclaw` CLI 命令（如 `openclaw gateway restart`）
- 禁止修改 `/opt/openclaw/` 系统文件
- 允许编辑工作区提示词文件（`SOUL.md`、`USER.md`、`AGENTS.md` 等）
