## 技术发现

飞书多维表格工具 `feishu_bitable_*` 没有创建字段的功能，但可以通过直接调用飞书 API 实现。

脚本位置：`~/.openclaw/workspace/create_bitable_fields.ts`

API 调用方式：
```typescript
POST https://open.feishu.cn/open-apis/bitable/v1/apps/{app_token}/tables/{table_id}/fields
Authorization: Bearer {tenant_access_token}
Body: { field_name: "字段名", type: 1 } // type: 1=文本, 3=单选, 7=复选框, 1001=创建时间
```
