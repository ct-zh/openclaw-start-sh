## 会话管理知识

用户询问 `/new` 斜杠命令是否会影响工作区 md 文件。回答：**不会**。`/new` 只清除会话上下文（JSONL 对话记录），工作区文件（MEMORY.md、SOUL.md 等）独立存储，不受影响。

## GitHub Token 配置

用户提供了 GitHub PAT token，已添加到 `~/.bashrc`：
```bash
export GITHUB_TOKEN="github_pat_..."
```

⚠️ 安全提醒：token 已在对话中暴露，建议用户在 GitHub 设置中删除并重新生成新 token。
