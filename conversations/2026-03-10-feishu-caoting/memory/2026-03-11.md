# 2026-03-11 Memory

## Feishu Bitable 调试

### 发现的问题

**bitable.ts 工具存在配置读取 bug**

位置：`/opt/openclaw/extensions/feishu/src/bitable.ts` 第 78-81 行

```typescript
const feishuCfg = api.config?.channels?.feishu as FeishuConfig | undefined;
if (!feishuCfg?.appId || !feishuCfg?.appSecret) {
  return; // 跳过注册！
}
```

问题：直接从 `channels.feishu.appId/appSecret` 读取，不支持多账户配置。

用户的配置结构：
```
channels.feishu.accounts.feishubot.appId  ✅ 存在
channels.feishu.appId                      ❌ 不存在
```

结果：bitable 工具因找不到凭证而未被注册，导致 `feishu_bitable_*` 系列工具不可用。

### 修复方案

修改 `bitable.ts`，使用 `listEnabledFeishuAccounts()` 替代直接读取顶层配置，参考其他工具（drive.ts, docx.ts）的实现方式。
